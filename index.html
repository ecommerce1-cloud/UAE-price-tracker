🚀 How to Setup Daily Scraping

🔑 Get Your Free ScraperAnt API Key

  1. Go to ScrapingAnt.com and sign up for a free account.
  2. Once registered, go to your Dashboard.
  3. Locate your API Key (a long string of characters). Copy it.
  4. Open your GitHub Repository settings, go to Secrets and variables > Actions.
  5. Create a new repository secret named: SCRAPINGANT_API_KEY and paste your key.

🔑 Copy Careem Session Token

Because Careem requires a mobile verification code, you need to copy your login token once from your web browser:

  1. Open a computer web browser (Chrome or Edge) and go to Careem.com.
  2. Log in with your phone number and enter the SMS verification code.
  3. Press F12 (or right-click anywhere and select Inspect) to open Developer Tools.
  4. Click on the Application tab (in Chrome) or Storage tab (in Firefox).
  5. In the left sidebar, expand Local Storage and click on https://www.careem.com.
  6. Look for the item named token or authToken. Copy the long value next to it.
  7. Add this value as a GitHub Repository secret named: CAREEM_TOKEN.

🌐 Enable GitHub Pages (To share your link)

  1. Push all these code files to a new repository on your GitHub account.
  2. Go to your repository's Settings > Pages.
  3. Under "Build and deployment", set the source to Deploy from a branch.
  4. Select the main branch and click Save.
  5. Wait 1 minute, and GitHub will provide you a link like: https://yourusername.github.io/your-repository. Copy and share it!
